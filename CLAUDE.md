# Regras de Animação
Toda animação Deste projeto deve obrigatoriamente usar:
1. **Lenis** para o gerenciamento de smooth scroll da página.
2. **GSAP** para todo movimento e interatividade (reveal de elementos, pin com ScrollTrigger, etc).

# Guidelines Estéticas (O Efeito Cinematográfico)
- Tensão Typography: Espere antes da primeira palavra aparecer.
- Pin Scroll com comportamento de câmera (aproximação e scale).
- Todo fade/move deve ter uma duração de no mínimo `0.6s` (frequentemente `1.2s` é o ideal para o tom cinematográfico).
- Utilize easing suave e arrastado (ex: `power3.out`, `expo.out`). Animações cinematográficas são lentas, controladas e intencionais, nunca abruptas.
